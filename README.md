## Reimagined World Selection

A brand new look for the World Selection Screen!

### Features:

*   Highly Configurable
*   Elements Animations
*   Integration with the KubeJS
*   Dynamic Updating Large Icons
*   Support For Custom Large Icons
*   Custom Lines With Your Custom Info

### Wiki:

If you want work with custom info, or know how does custom large icons work, check out mod **[wiki](https://github.com/Inf2z/reimagined-world-selection_1.21.1/wiki)**!
